Quick Start

Up and running in minutes

Install shelf and process your first book scan.

# Clone and install
$ git clone https://github.com/jackzampolin/shelf.git
$ cd shelf && uv venv && source .venv/bin/activate
$ uv pip install -e .
# Add your OpenRouter API key
$ echo "OPENROUTER_API_KEY=sk-..." > .env
# Ingest a book scan (folder of page images)
$ shelf ingest /path/to/scans --title "My Book"
# Process the book through all stages
$ shelf book my-book process
# Launch the web UI to review
$ shelf serve --port 8080
