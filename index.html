Quick Start

Get up and running in minutes

# Clone the repository
git clone https://github.com/jackzampolin/shelf.git
cd shelf
# Install dependencies
pip install -r requirements.txt
# Run the pipeline
python -m shelf process --input scanned_book.pdf
